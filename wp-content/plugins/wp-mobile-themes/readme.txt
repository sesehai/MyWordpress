Display a mobile theme when the website is viewed with a mobile browser.

== Description ==

Allows you select another theme that will be sent to mobile users.

为手机用户选择另一个主题.

**Supported Languages:**

* US English/en_US (default)
* 简体中文/zh_CN ([mg12](http://www.neoease.com/))

**Demo:**

[http://www.neoease.com/](http://www.neoease.com/)

== Installation ==

1. Unzip archive to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Appearance->Mobile Themes' to select a mobile theme.
