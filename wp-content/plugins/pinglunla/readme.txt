== Installation ==

1. 安装步骤：搜索并安装插件→ 启用插件→ 登陆（注册）评论啦→ 高级选项导入评论
2. 如果无法在线安装插件，请下载本插件，解压至`/wp-content/plugins/`文件夹即可启用本插件
3. sinaapp用户请使用Json导出数据，再到评论啦官网导入数据实现评论导入
3. 如果安装插件遇到问题请联系评论啦技术获得支持（QQ：`2407672560`），更多帮助：http://pinglun.la/help
